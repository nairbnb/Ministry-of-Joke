## Repository Conventions

**Branches**

- `main` — protected; merge via pull request only
- Feature work: `feature/short-description`
- Bug fixes: `fix/short-description`

**Pull Requests**

- At least one teammate must review and approve before merge
- PR description must reference the ICEX or assignment it addresses
- CI must pass before merge

**Commits**

- Imperative present tense: "Add joke submission endpoint" not "Added..."
- Reference the assignment (e.g. `ICEX_MJ03`) when applicable
